
# NEXORA Custom Announcement Mode

هذا الباتش يفعّل خيار:
`تصميم مخصص / نص حر`

## شنو يشتغل؟

- الضغط على `تصميم مخصص / نص حر` يحول نوع القالب إلى `custom`.
- لا يتم ملء العنوان والوصف من قالب جاهز.
- تكتب العنوان والوصف بنفسك.
- المعاينة تتغير مباشرة حسب النص الذي تكتبه.
- الحفظ في قاعدة البيانات يكون كإعلان مخصص.
- بدون LocalStorage.

## التشغيل

```powershell
cd C:\Users\Lenovo\SmartAttendance
powershell -ExecutionPolicy Bypass -File .\Apply_NEXORA_Custom_Announcement_Mode.ps1
```

بعد التشغيل:

```text
Ctrl + F5
```
