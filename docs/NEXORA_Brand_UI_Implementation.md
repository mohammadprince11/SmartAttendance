# NEXORA Brand UI Implementation

هذا الباتش يطبق هوية NEXORA على واجهة النظام.

## التغييرات
1. تغيير واجهة Layout إلى NEXORA.
2. إضافة شعار SVG:
   - wwwroot/brand/nexora-symbol.svg
   - wwwroot/brand/nexora-logo.svg
3. إضافة ملف CSS رئيسي:
   - wwwroot/css/nexora-brand.css
4. إضافة JavaScript للـ Sidebar والـ Active Links:
   - wwwroot/js/nexora-brand.js
5. إضافة صفحة عرض الهوية:
   - /Brand
6. إضافة Dashboard style layer:
   - wwwroot/css/dashboard-nexora.css

## ألوان الهوية
- Midnight Navy: #07121F
- Deep Space: #0F2233
- Electric Teal: #12D9E3
- Champagne Gold: #D7B26D
- Soft Silver: #E6EAEE

## قاعدة مهمة
لم يتم استخدام صور HR أو بصمة أو ساعة داخل التصميم.
التصميم متعمد أن يظهر كمنصة تقنية ذكية وشاملة.
